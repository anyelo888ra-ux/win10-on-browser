# Windows 10 en el Navegador 💻

Un simulador web ligero de Windows 10 desarrollado con **Node.js**, **Express**, **HTML5** y **CSS3**. Diseñado para ejecutarse completamente en la nube sin consumir almacenamiento local.

## Características
- 🖥️ Escritorio interactivo con fondo clásico.
- 🗔 Ventanas flotantes que consultan una API en tiempo real.
- 🕒 Reloj del sistema funcional sincronizado.
- ☁️ Listo para desplegar 24/7 en Render.
